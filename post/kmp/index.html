KMP

2022-03-21 2 min read


学习一下kmp
例题：
给定一个模式串 S，以及一个模板串 P，所有字符串中只包含大小写英文字母以及阿拉伯数字。
模板串 P 在模式串 S 中多次作为子串出现。
求出模板串 P 在模式串 S 中所有出现的位置的起始下标。

输入格式
第一行输入整数 N，表示字符串 P 的长度。
第二行输入字符串 P。
第三行输入整数 M，表示字符串 S 的长度。
第四行输入字符串 S。

输出格式
共一行，输出所有出现位置的起始下标（下标从 0 开始计数），整数之间用空格隔开。

数据范围
1≤N≤10^5
1≤M≤10^6

#include
using namespace std;
const int M=100004;
int n,m;
string s,p;
int ne[M];
int main()
{
cin>>n>>p>>m>>s;
for (int i=1,j=0;i<n;i++)
{
while( j && p[i] != p[j] )
j = ne[j-1];
if( p[i] == p[j] )
j++;
ne[i] = j;
}
int cnt = 0;
for( int i = 0 , j = 0 ; i < m ; i ++ )
{
while( j && s[i] != p[j] )
j = ne[j-1];
if( s[i] == p[j] )
j++;
if( j == n)
{
j = ne[j-1];
cout<<i-n+1<<" ";
}
}
return 0;
}
